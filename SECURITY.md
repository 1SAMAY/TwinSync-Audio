# Security Policy

## Supported Versions

| Version | Supported |
| --- | --- |
| 0.1.x | Yes |

## Reporting A Vulnerability

Please report security issues through GitHub:

https://github.com/1SAMAY/TwinSync-Audio/issues

If the repository enables private vulnerability reporting later, prefer that channel.

Include:

- TwinSync Audio version
- Windows version
- Steps to reproduce
- Impact
- Relevant local logs with personal information removed
- Whether the issue requires a malicious audio device, local user access, or a crafted file

Do not post secrets, private keys, or sensitive personal information in public issues.
