# Code of Conduct

TwinSync Audio contributors are expected to keep project spaces respectful, practical, and focused on building a useful local Windows audio tool.

## Expected Behavior

- Be respectful of different experience levels.
- Assume good intent.
- Keep feedback specific and actionable.
- Avoid harassment, insults, threats, and personal attacks.
- Do not publish private information without permission.

## Reporting

Report conduct concerns through GitHub Issues or the repository owner's preferred contact channel.
