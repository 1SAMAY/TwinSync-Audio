# TwinSync Audio v0.2.0 User Guide

1. Connect both outputs in Windows Sound settings.
2. Select Primary and Secondary in TwinSync.
3. Use Test Primary and Test Secondary.
4. Start routing from the permanent HUD.
5. Increase manual trim only on the speaker that sounds early, or select a microphone and run acoustic calibration.
6. Save the result as a profile.

The HUD always shows session, selected outputs, health, master volume, Start, Stop, Emergency Stop, and Settings. If a selected Windows-default speaker is not controlled by TwinSync in guarded loopback mode, its name and the required virtual-source alternative are displayed.

Automatic calibration applies only when repeated measurements agree and correlation confidence is acceptable. Guided calibration remains available without a microphone.

Developer diagnostics can be exported locally. The export excludes device IDs/names, usernames, recordings, event messages, and full error text.
