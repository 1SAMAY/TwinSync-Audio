# Support

Use GitHub Issues for bugs, feature requests, and device compatibility reports:

https://github.com/1SAMAY/TwinSync-Audio/issues

## Useful Information For Bug Reports

- TwinSync Audio version
- Windows version
- Speaker models
- Bluetooth adapter or audio interface
- Whether each speaker is Bluetooth, USB, HDMI, or analog
- Steps to reproduce
- Expected behavior
- Actual behavior
- Screenshots when useful
- Relevant logs from `%LOCALAPPDATA%\TwinSyncAudio\logs`

Do not share private recordings, passwords, tokens, or sensitive personal data.

## Synchronization Reports

For sync quality issues, include the observed delay, codec if Windows exposes it, whether the speakers reconnect cleanly, and whether the issue changes after restarting Bluetooth.
